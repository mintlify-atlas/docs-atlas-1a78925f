## Overview

The Alert Types API allows you to configure alert types, including their names, descriptions, priority levels, and responsible parties. This API uses GraphQL queries and mutations.

<Note>
  All alert type operations use the `/api/alerts/graphql` endpoint.
</Note>

## Priority Level Mapping

From `~/workspace/source/src/pages/Dashboard.tsx:125-136`:

```typescript
const getPriorityId = (priority: string): number => {
  // If priority is a number (ID), return it
  const id = parseInt(priority);
  if (!isNaN(id)) return id;
  
  // Fallback for compatibility
  const p = priority.toLowerCase();
  if (p.includes("alta") || p === "high") return 3;
  if (p.includes("media") || p === "medium") return 2;
  if (p.includes("baja") || p === "low") return 1;
  return 1;
};
```
