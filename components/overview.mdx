## Introduction

FleetGuard360 uses a modern component architecture built on top of **shadcn/ui** and **Radix UI** primitives. The application features 48+ pre-built, accessible UI components combined with custom business components for fleet monitoring.

## Component Library

The component system is organized into two main categories:

## Directory Structure

All components are located in the `src/components/` directory:

```
src/components/

│   └── ...
```

### NotificationBell Component

The `NotificationBell` component displays real-time alerts using WebSocket:

```tsx src/components/NotificationBell.tsx
import { useState } from "react";
import { Bell } from "lucide-react";
import { Button } from "@/components/ui/button";
import { Badge } from "@/components/ui/badge";
import { Popover, PopoverContent, PopoverTrigger } from "@/components/ui/popover";
import { ScrollArea } from "@/components/ui/scroll-area";
import { useWebSocketNotifications } from "@/hooks/use-websocket-notifications";
import { useToast } from "@/hooks/use-toast";

export const NotificationBell = () => {
  const { toast } = useToast();
  const [isOpen, setIsOpen] = useState(false);

  const { notifications, isConnected, clearNotifications } = 
    useWebSocketNotifications({
      onNotification: (notification) => {
        toast({
          title: "Nueva Alerta",
          description: `${notification.alertType} - ${notification.generatingUnit}`,
          variant: notification.priority?.toLowerCase().includes("alta") 
            ? "destructive" 
            : "default",
        });
      },
    });

  const unreadCount = notifications.length;

  return (
    <Popover open={isOpen} onOpenChange={setIsOpen}>
      <PopoverTrigger asChild>
        <Button variant="ghost" size="icon" className="relative">
          <Bell className="h-5 w-5" />
          {unreadCount > 0 && (
            <Badge className="absolute -top-1 -right-1">
              {unreadCount > 9 ? "9+" : unreadCount}
            </Badge>
          )}
        </Button>
      </PopoverTrigger>
      <PopoverContent className="w-80">
        {/* Notification list... */}
      </PopoverContent>
    </Popover>
  );
};
```

**Key Features:**
- Real-time WebSocket integration
- Unread notification count badge
- Toast notifications for new alerts
- Priority-based styling
- Connection status indicator

## Component Composition

FleetGuard360 components follow composition patterns:

<CodeGroup>
```tsx Card Composition
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";

<Card>
  <CardHeader>
    <CardTitle>Alert Statistics</CardTitle>
  </CardHeader>
  <CardContent>
    {/* Content */}
  </CardContent>
</Card>
```

```tsx Dialog Composition
import {
  Dialog,
  DialogContent,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@/components/ui/dialog";

<Dialog>
  <DialogTrigger asChild>
    <Button>Open</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Title</DialogTitle>
    </DialogHeader>
    {/* Content */}
  </DialogContent>
</Dialog>
```
</CodeGroup>
