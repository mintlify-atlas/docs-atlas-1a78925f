## CSV Export Functionality

### Export Query

From `Estadisticas.tsx:22-26`, the system uses a GraphQL query to get the CSV download URL:

```graphql
query {
  exportAlertsCsvUrl
}
```

### Implementation

From `Estadisticas.tsx:13-61`, the export functionality:

```typescript
const [isExporting, setIsExporting] = useState(false);
const { fetchGraphQL } = useGraphQL();

const handleExportCSV = async () => {
  setIsExporting(true);
  try {
    toast({
      title: "Preparando exportación",
      description: "Generando archivo CSV...",
    });

    // Fetch the download URL
    const query = `
      query {
        exportAlertsCsvUrl
      }
    `;

    const result = await fetchGraphQL<{ exportAlertsCsvUrl: string }>(
      "/panel/graphql",
      query
    );

    const downloadUrl = result.exportAlertsCsvUrl;

    // Create a download link
    const link = document.createElement("a");
    link.href = downloadUrl;
    link.download = `alertas_${new Date().toISOString().split('T')[0]}.csv`;
    link.target = "_blank";
    document.body.appendChild(link);
    link.click();
    document.body.removeChild(link);

    toast({
      title: "Exportación exitosa",
      description: "El archivo CSV ha sido descargado correctamente",
    });
    
  } catch (error) {
    console.error("Error exportando CSV:", error);
    toast({
      title: "Error",
      description: "No se pudo exportar el archivo CSV",
      variant: "destructive",
    });
  } finally {
    setIsExporting(false);
  }
};
```

## CSV File Format

The exported CSV file includes all alert history data:

```csv
id,alertType,responsible,priority,driver,generatingUnit,state,generationDate
123,Exceso de velocidad,Área de monitoreo,Alta,Carlos Sanchez,Camión-123,En proceso,2026-03-04T10:30:00Z
124,Frenado brusco,Mantenimiento,Media,Ana Gómez,Furgoneta-456,Resuelto,2026-03-04T11:15:00Z
```

### CSV Fields

<ParamField path="id" type="string">

</ParamField>

<ParamField path="alertType" type="string">
  Type of alert generated
</ParamField>

<ParamField path="responsible" type="string">
  Responsible party or department
</ParamField>

<ParamField path="priority" type="string">

</ParamField>

<ParamField path="driver" type="string">
  Driver name associated with the alert
</ParamField>

<ParamField path="generatingUnit" type="string">
  Vehicle or unit that generated the alert
</ParamField>

<ParamField path="state" type="string">
  Current state: En espera, En proceso, or Resuelto
</ParamField>

<ParamField path="generationDate" type="string">
  ISO 8601 timestamp of alert generation
</ParamField>

1. **User clicks export button**
   ```typescript

   ```

2. **Show loading state**

3. **Query GraphQL for download URL**

4. **Create temporary download link**
   ```typescript

   link.href = downloadUrl;
   link.download = `alertas_${new Date().toISOString().split('T')[0]}.csv`;
   link.target = "_blank";
   ```

5. **Trigger download**
   ```typescript
   document.body.appendChild(link);
   link.click();
   ```

6. **Show success/error message**
   ```typescript

   ```

## Power BI Customization

### Obtaining Your Report URL

1. **Publish to Power BI Service**
   - Create your report in Power BI Desktop
   - Publish to Power BI Service workspace

2. **Share Report**
   - Open report in Power BI Service
   - Click "File" → "Embed report" → "Website or portal"
   - Copy the secure embed URL

3. **Update Component**
   ```tsx
   <iframe 
     src="YOUR_POWERBI_EMBED_URL_HERE"
     // ... other props
   />
   ```

<Warning>
  Ensure your Power BI report permissions are correctly configured to allow embedding.
</Warning>

## Troubleshooting

### Power BI Dashboard Not Loading

1. **Check URL validity** - Verify the embed URL is correct and accessible
2. **CORS issues** - Power BI must allow embedding from your domain
3. **Permission errors** - Ensure the report has public sharing enabled or appropriate permissions

### CSV Export Fails

1. **Check authentication** - Ensure JWT token is valid
2. **Backend availability** - Verify the `/panel/graphql` endpoint is accessible
3. **Browser blocking** - Check if popup blockers are preventing the download
4. **Network errors** - Inspect browser console for network errors

### Export Button Stuck Loading

```typescript
// Always reset loading state in finally block
try {
  // ... export logic
} catch (error) {
  // ... error handling
} finally {
  setIsExporting(false); // Always reset
}
```

## Security Considerations

<Warning>
  **Public embedding** - The current Power BI URL is publicly accessible. For production, use row-level security (RLS) or embed tokens.
</Warning>

<Note>
  **CSV data** - The export URL is obtained via authenticated GraphQL request, ensuring only authorized users can export data.
</Note>

<Note>
  **JWT validation** - All GraphQL requests are authenticated with JWT Bearer tokens.
