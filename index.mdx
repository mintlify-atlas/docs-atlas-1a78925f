<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with FleetGuard360 in minutes</p>

  <Steps>
    <Step title="Clone the Repository">
      Clone the FleetGuard360 frontend repository to your local machine:
      
      ```bash
      git clone https://github.com/CodeFactory-FleetGuard360-EV04/Frontend.git
      cd Frontend
      ```
    </Step>
    
    <Step title="Install Dependencies">
      Install all required packages using npm or your preferred package manager:
      
      ```bash
      npm install
      ```
      
      <Note>The project uses React 18, Vite, TypeScript, and shadcn/ui components.</Note>
    </Step>
    
    <Step title="Configure Environment">
      The application uses proxy configuration for API requests. The backend API is automatically proxied through Vite:
      
      ```typescript
      // vite.config.ts
      proxy: {
        "/api": {
          target: "http://fleetguard-api.ddns.net",
          changeOrigin: true
        }
      }
      ```
    </Step>
    
    <Step title="Start Development Server">
      Launch the development server and access the application:
      
      ```bash
      npm run dev
      ```
      
      The application will be available at `http://localhost:3000`. Log in with your operator credentials to access the dashboard.
    </Step>
  </Steps>
</div>
